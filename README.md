# Time-Zone-Program

## Description
- A C#/.NET program where the user selects a city and the applications responds with the correct time zone for that city.

## Technologies Used
- C#
- .NET
- Visual Studio
  
## Features
- Time zone calculation.
- User input validation.
- Confirm & Exit Buttons.
  
## What I Learned
- How to write if/else statements in C#.
- Creating event handler for "Ok" button in C#.
- Creating event handler for "Exit" button in C#.
- Creating list box and adding cities within it in C#.
